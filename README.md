**最后给大家推荐一些学习资源**  

[《Angular权威教程》](http://www.ituring.com.cn/book/1874)

官方文档，至少要把TS的基本语法要去掌握了。

[Rxjs官方文档中文翻译](https://www.gitbook.com/book/buctwbzs/rxjs/details)  
翻译质量不错，值得一看。
